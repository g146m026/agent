# Conformité Copilot – Instructions, Skills, Prompts

Tous les agents, skills et prompts de ce dossier sont structurés selon les best practices Microsoft Copilot (2026) :

- Instructions en Markdown, sections claires (objectif, règles, skills, workflow, contrat de sortie, exemples, self-check)
- Langage actionnable, tâches atomiques, workflow séquencé
- Contrat de sortie explicite (format, ton, niveau de détail)
- Self-evaluation gate systématique
- Header de stabilité si besoin
- Patterns et templates réutilisables dans `prompts/templates/`
- Audit régulier de chaque agent/skill pour garantir la conformité

Voir `REFERENCES.md` pour la source officielle et les liens utiles.
