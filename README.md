# Agent Workspace

Ce dossier contient la logique, les skills, prompts, scripts, tests et documentation pour les agents racine.

## Structure
- `skills/` : capacités et modules agents
- `prompts/` : templates et exemples d’usage
- `scripts/` : helpers, outils de migration, validation
- `tests/` : tests unitaires et d’intégration (segmentés par type)
- `.github/workflows/` : CI, lint, validation
- `doc/` : documentation, guides, checklists
- `archives/` : anciens fichiers pour traçabilité

Voir `doc/` pour la documentation détaillée, la gestion des secrets/env, et le plan de migration.
