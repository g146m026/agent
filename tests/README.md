# Tests dans agent/tests/

Seuls les tests généraux, d’intégration inter-module, ou liés à l’intelligence/validation des agents sont conservés ici.

Les tests spécifiques à chaque module ou application restent dans leur dossier d’origine (ex : `jukebox/tests/`, `girard-hub/tests/`, etc.).

Pour toute nouvelle contribution, placer les tests au bon endroit selon cette règle.
